## 小红称体重

### 问题描述
小红每天都会称体重，连续称`n`天。从第三天开始，如果有一天体重不小于 `alpha`，且前一天的体重小于 `alpha`，那么小红这天就会不开心。
小红想知道，她一共会有多少天不开心？

**输入：**
- `n`: 天数，2 ≤ n ≤ 100
- `alpha`: 体重阈值，1 ≤ alpha ≤ 100
- `weights`: 每天的体重，1 ≤ weights ≤ 100

**输出：**
- 一个整数，代表不开心的天数。

### 测试样例    

***示例1***
>输入： `n = 5`, `alpha = 53`, `xi = [50, 52, 55, 53, 55]`
>输出：`1`
>说明：只有第三天会不开心。
