## 赝品

### 问题描述
有一批型号相同的商品中混入了若干个型号不同的商品，请你找出这些黑品并将它们按型号从小到大输出。
保证每个黑品的型号只出现了一次。

**输入**
- `n`: 一个正整数，代表商品的总数量，3 ≤ `n` ≤ 10^5。
- `arr`: 含有`n`个正整数，代表每个商品的型号，1 ≤ `arr[i]` ≤ 10^9。
- 保证至少存在一个黑品，保证黑品的出现次数不小于2次。

**输出**    
- 输出一个数组，包含`k`个正整数，代表每个黑品的型号。

### 测试样例

***示例1***
>输入：`n = 5`, `arr = [2, 5, 3, 2, 2]`
>输出：`[3, 5]`
>说明：型号2号是正品，其余的为黑品。

***示例2***
>输入：`n = 4`, `arr = [9, 9, 2, 9]`
>输出：`[2]`
